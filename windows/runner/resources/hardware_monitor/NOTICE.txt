Hardware monitor runtime

This directory contains the PowerShell probe and .NET assemblies used to read
CPU temperature sensors on Windows.

Primary library:
- LibreHardwareMonitorLib 0.9.6, MPL-2.0
  https://github.com/LibreHardwareMonitor/LibreHardwareMonitor

Runtime dependencies are copied from their NuGet packages for the net472
Windows runtime where available.
